Description of the Task 3: ATM System

This project was developed as Task 3 during my internship at CodSoft from 10 January 2026 to 10 February 2026.

The objective of this task was to design a console-based Java ATM simulation system that allows users to securely authenticate using a PIN and perform common banking operations such as checking balance, depositing money, withdrawing cash, and viewing transaction history.

What the Program Does

PIN-based user authentication with limited attempts
Displays ATM-style interactive menu
Allows balance inquiry
Supports deposit transactions with validation
Supports withdrawal with insufficient balance checks
Stores and displays transaction history
Input validation for numeric entries
Allows safe exit from the system
